# Plot method for `MRpms_modas` class with dimension `1`.

Plot method for `MRpms_modas` objects, which are computed from a `PMS`
call or, a more bare-bones version, `PMSc` call.

## Usage

``` r
# S3 method for class 'MRpms_modas'
plot(x, data, ...)
```

## Arguments

- x:

  An `MRpms_modas` object to plot.

- data:

  The data from which the modes were computed.

- ...:

  Other parameters

## Value

A plot with both the `data` and the modes.
