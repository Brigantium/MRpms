# Two sines shifted in Y axis.

Two sines shifted in Y axis.

## Usage

``` r
twosines
```

## Format

An object of class `matrix` (inherits from `array`) with 200 rows and 2
columns.

## Details

The 200 points from two sines with a certain normal error,

- 100 from the usual sine ploted between 0 and 10 (in radians),

- 100 from a shifted sine 3 points lower between the same support.

In both cases, error follows a certain \\\mathcal N(0,0.3)\\.

## Examples

``` r
plot(twosines)

```
