# Print an `MRpms_conf` object

A generic function to print an `MRpms_conf` object, which are computed
from a `ConfPMS` call.

## Usage

``` r
# S3 method for class 'MRpms_conf'
print(x, ...)
```

## Arguments

- x:

  An `MRpms_conf` object.

- ...:

  Other arguments. Currently, nothing more is avaliable.

## Value

An index of contents.
