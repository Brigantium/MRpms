# Print useful information of an `MRpms_pred` object

A generic function to summarize useful information of an `MRpms_pred`
object, which are computed from a `PredPMS` call.

## Usage

``` r
# S3 method for class 'MRpms_pred'
summary(object, ...)
```

## Arguments

- object:

  An `MRpms_pred` object.

- ...:

  Other arguments. Currently, nothing more is avaliable.

## Value

The summarize of the `MRpms_modas` object, plus estimated radius for
uniform prediction set and volume of said region.
