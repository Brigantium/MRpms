# Print an `MRpms_modas` object

A generic function to print an `MRpms_modas` object, which are computed
from a `PMS` cal or, a more bare-bones version, `PMSc` call.

## Usage

``` r
# S3 method for class 'MRpms_modas'
print(x, ...)
```

## Arguments

- x:

  An `MRpms_modas` object.

- ...:

  Other arguments. Currently, nothing more is avaliable.

## Value

The modes in every covariable point from `x.malla`, which is an
attribute of the `MRpms_modas`.
