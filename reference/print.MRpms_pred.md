# Print an `MRpms_pred` object

A generic function to print an `MRpms_pred` object, which are computed
from a `PredPMS` call.

## Usage

``` r
# S3 method for class 'MRpms_pred'
print(x, ...)
```

## Arguments

- x:

  An `MRpms_pred` object.

- ...:

  Other arguments. Currently, nothing more is avaliable.

## Value

The radius of the uniform prediction interval and the first modes.
